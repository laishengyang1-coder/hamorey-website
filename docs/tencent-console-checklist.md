# 腾讯云控制台待办清单

这份清单只记录需要在腾讯云网页里点的事项。能用脚本完成的内容放在 `scripts/` 里。

## 1. TencentDB MySQL

状态：需要购买，等你确认套餐。

建议配置：

- 地域：广州，和轻量服务器一致
- 版本：MySQL 8.0
- 架构：正式生产优先选双节点；预算紧张可以先单节点，但后面要升级
- 规格：1C2G 起步；如果演示和门店逐步增长，2C4G 更稳
- 存储：50GB 起步
- 数据库名：`hamorey`
- 数据库用户：`hamorey`

买好后需要拿到：

- 内网地址
- 端口，通常是 `3306`
- 用户名
- 密码

这些内容后面写到服务器的 `/etc/hamorey/api.env`，不要发到公开聊天或 GitHub。

## 2. COS 存储桶

状态：你已经买了 COS 资源包，但还需要确认是否已创建业务桶。

建议创建：

- 桶名：`hamorey-assets`
- 地域：广州
- 访问权限：私有读写
- 版本控制：先不开
- 静态网站：先不开

用途：

- 施工照片
- 质保证书 PDF
- 后台导出文件
- 积分商城商品图片

## 3. COS 最小权限密钥

不要直接使用主账号永久密钥。建议创建子用户：

- 子用户名称：`hamorey-server`
- 访问方式：编程访问
- 权限：只给 `hamorey-assets` 这个桶读写权限

拿到后只保存：

- SecretId
- SecretKey

这些内容只写进服务器 `/etc/hamorey/api.env`，不要提交到 GitHub。
